Membuat Game Lato - Lato Sederhana Menggunakan HTML, CSS, dan JavaScript

Kalian pasti tahu kan, permainan anak - anak bernama "lato lato"?. Apalagi kepopuleran benda ini meningkat bahkan viral pada desember 2022 lalu. permainan berupa 2 buah bola yang diikatkan pada sebuah tali yang saling menyambung ini sangatlah seru untuk dimainkan. Mulai dari anak - anak bahkan orang dewasa sekalipun mencoba untuk memainkan permainan tersebut. nah kali ini kita akan membuat game lato lato sederhana menggunakan HTML, CSS, dan JavaScript! kuy simak artikel nya dan jangan lupa untuk share ke temen temen kamu.
Pertama, kamu buat sebuah file bernama index.html lalu isi dengan kode dibawah ini :

<div id="container"> <h1>Lato Lato Game</h1> <h1 id="score"></h1> <div id="bola"> <div id="garis1"></div> <div id="garis2"></div> <div id="bola1"></div> <div id="bola2"></div> </div> </div>

kita styling lato lato hingga berbentuk bulat, kalian bisa menambahkan file CSS/menggunakan CSS eksternal atau disimpan pada bagian head :


* { margin: 0; padding: 0; touch-action: manipulation; font-family: Helvetica; } #container { width: 25rem; margin: auto; text-align: center; } #bola { display: grid; justify-content: space-between; position: relative; grid-template-areas: "tali tali" "bola bola"; } #garis1, #garis2 { height: 30rem; width: 5px; background-color: #000; justify-self: center; } #garis1 { transform: rotate(17deg); margin-right: -160px; } #garis2 { transform: rotate(-17deg); margin-left: -160px; } #bola1, #bola2 { width: 100px; height: 100px; background-color: #000; border-radius: 100%; margin-top: -1rem; } .space { justify-content: center; } .rotatekanan { transform: rotate(6deg); margin-right: -50px; } .rotatekiri{ transform: rotate(-6deg); margin-left: -50px; } h1 { margin: 1rem; } #score { text-align: center; margin-top: 1rem; } #boom { position: absolute; bottom: 5 !important; width: 5rem; text-align: center; display: none; transform: translateX(200%); filter: drop-shadow(0px 0px 20px #ff2b05); }

Setelah itu, kita tambahkan JavaScript pada file html agar lato lato bisa dimainkan. sama seperti css, kalian bisa membuat file eksternal atau membuat tag script pada bagian akhir body :

var x = 0; window.addEventListener('touchstart', function(){ document.getElementsByClassName('bola')[0].classList.add('space') document.getElementsByClassName('garis1')[0].classList.add('rotatekanan') document.getElementsByClassName('garis2')[0].classList.add('rotatekiri') }) window.addEventListener('touchend', function(){ document.getElementsByClassName('bola')[0].classList.remove('space') document.getElementsByClassName('garis1')[0].classList.remove('rotatekanan') document.getElementsByClassName('garis2')[0].classList.remove('rotatekiri') x += 1; document.getElementById('score').innerHTML = x - 1 })

Hore!!! Selamat! kamu berhasil membuat sebuah game sederhana menggunakan HTML, CSS, dan JavaScript. untuk preview nya kamu bisa cek disini. Jika ada pertanyaan dan saran silahkan hubungi pada field yang telah disediakan. Terimakasih!!!

Top 5 Bahasa Pemrograman yang Wajib Kamu Pelajari!


Foto : Unsplash

Hi guys!, kali ini saya akan membahas tentang 5 bahasa pemrograman yang wajib banget kamu pelajarin menurut pendapat saya. Kebanyakan bahasa pemrograman ini banyak sekali dicari di dunia kerja. Penasaran apa aja bahasa pemrograman tersebut? yuk simak artikel ini. Jangan lupa share ke temen kamu biar mereka tahu!


1. JavaScript


JavaScript atau disingkat JS merupakan salah satu bahasa pemrograman terpopuler di dunia. bagaimana tidak? bahasa ini memiliki fungsi yang cukup lengkap, mulai dari pembuatan aplikasi, sampai pembuatan website! ya, JavaScript ini merupakan otak dari pembuatan website yang responsif. Jadi kalau kamu mau jadi Web Developer, wajib banget nih tau tentang JavaScript!


2. Python


Siapa sih yang nggak tahu si ular sanca ini? wkwkwk. Python merupakan salah satu bahasa pemrograman paling populer di dunia. Sintaksnya yang cukup mudah dimengerti untuk pemula membuatnya banyak diminati oleh kalangan para programmer. selain itu, Python memiliki banyak sekali fungsi mulai dari pembuatan website, aplikasi, bahkan Artificial Intelligence (AI). Kalau kamu mau bikin Machine Learning, kamu bisa nih pakai Python!


3. C++


C++ atau dibaca Cpp merupakan bahasa pemrograman hasil pengembangan dari bahasa C. Kemampuannya yang sangat cepat dalam menjalankan program membuat C++ ini banyak digunakan di dalam pembuatan Game! karena pada umumnya, game membutuhkan respons yang cepat. nah c++ ini sangat cocok nih buat jadi inti program dari sebuah game. oh iya, jika kamu mau, kamu juga bisa mempelajari bahasa C yang merupakan bapak dari segala bahasa :D, sering digunakan untuk pembuatan sistem operasi seperti Windows dan Linux


4. Kotlin


Kotlin merupakan bahasa pemrograman yang dibuat untuk membuat aplikasi android. Bahasa ini disupport langsung oleh Google dalam pembuatan sebuah aplikasi. Bahkan jika kamu akan menginstall Android Studio, bahasa pemrograman yang direkomendasikan adalah kotlin


5. PHP : Hypertext Preprocessor


Yang terakhir ada php nih. PHP ini sering digunakan untuk pembuatan website agar sebuah website menjadi dinamis. maksudnya adalah website tersebut dapat berinteraksi/memanipulasi dengan database seperti SQL atau noSQL. Kalau kamu menemukan sebuah website dengan sistem login, kemungkinan website tersebut menggunakan bahasa pemrograman php


(BONUS) 6. Swift


Satu lagi nih lumayan nambah nambah hehe. Swift merupakan bahasa pemrograman yang digunakan untuk pembuatan aplikasi di lingkungan iOS. Sebelumnya bahasa yang digunakan adalah objective-C namun perusahaan apple kemudian membuat bahasa pemrograman yang lebih simple khusus untuk lingkungan buah apel. pantesan nih keamanannya terjamin!


jadi itulah beberapa bahasa pemrograman yang wajib banget kamu pelajarin di 2023 ini. semoga bermanfaat dan mohon maaf bila ada kesalahan dalam penulisan maupun jika ada koreksi tolong diberitahu. Terimakasih!!!

Cara Membuat Efek Light Trails di HP


Foto : Unsplash

Kamu pasti bingung gimana supaya street photography kamu lebih unik untuk diupload di media sosial? dan kamu pasti sering melihat efek cahaya yang seolah bergerak melesat dalam sebuah foto? nah efek tersebut dinamakan light trails atau lintasan cahaya. disini, saya akan membagikan sebuah tips & trik bagaimana cara membuat efek lighttrails cuma pake hp doang! yuk simak penjelasannya! Jangan lupa share ke temen kamu biar mereka tahu! (penangkapan gambar menggunakan perangkat kamera professional mungkin hampir serupa)


jadi yang pertama pastikan bahwa smartphone kamu memiliki mode pro yang lengkap seperti ISO, Shutterspeed, dll. jika hp kamu tidak memiliki fitur tersebut, cobalah untuk menginstall aplikasi pihak ketiga seperti Google Cam/Gcam dengan mengetik pada play store "GCamator" lalu tekan install dan ikuti lamgkah langkah yang diberikan.


kedua, buka mode pro pada aplikasi kamera


kemudian yang ketiga, atur shutter speed menjadi 5 detik atau sesuaikan dengan kebutuhan masing masing(semakin lama, cahaya yang dihasilkan akan semakin panjang, namun lihat ketentuan pada langkah ke 4!)


keempat, atur ISO menjadi yang terendah. untuk pengguna kamera professional, bisa menggunakan ND filter untuk membuat lensa lebih gelap. Hal ini dilakukan karena proses penangkapan cahaya akan menghasilkan pantulan yang sangat banyak sehingga harus diminimalisir dengan menurunkan ISO. semakin lama durasi shutterspeed, maka akan semakin terang gambar yang dihasilkan, jadi sesuaikan ISO dengan Shutterspeed agar foto tidak menjadi berwarna putih


selanjutnya yang kelima, simpan hp/kamera di tempat yang diam seperti ditempatkan pada tripod. diusahakan hp/kamera tersebut tidak goyang sedikit pun karena akan sangat mempengaruhi pada hasil gambar


keenam, tekan tombol untuk menangkap gambar lalu tunggu selama durasi shutterspeed yang telah diatur sebelumnya


ta - da! kamu berhasil membuat efek light trails cuma pakai hp!


share konten ini ke temen temen kamu! Terimakasih!
